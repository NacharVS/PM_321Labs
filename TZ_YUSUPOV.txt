Приложение на Blazor.
Общие:
Цветовая палитра белый/черный/синий.
Возможность регистрации как сотрудник.
Возможность регистрации клиента.
Возможность забронировать номер клиенту.
Возможность добавить новый номер.

Начальный экран/Экран загрузки.
При старте приложения страница авторизации.

Форма регистрации сотрудника:
Текстовые поля:
1.1 Для ввода ФИО.
1.2 Для ввода номера телефона
1.3 Для ввода  E-Mail адреса
Кнопки:
2.1 Для подтверждения регистрации

Домашняя страница (Для сотрудника).
	Информация о сотруднике:
1.1 Фамилия, имя, отчество
1.2 Дата рождения
1.3 Мобильный телефон
1.4. E-Mail
1.5 Логотип 
Левая панель:
	Функциональная часть:
	Вкладка бранирование, с вводом нового бронированием и редактированием старого.
	Вид вкладки: таблица со всеми бронированными номерами, кнопка редактирования, которая открывает форму редактирования.
	Форма Редактирования:
	1. Поле Номер комнаты.
	2. Поле Клиент(select с выбором клиента)
	3. Поле Дата заезда
	4. Поле Дата выезда
	5. Поле кол-во людей.
	6. Поле Общая сумма проживания.

	Вкладка Клиенты.
	Вид вкладки: таблица со всем клиентами которые проживали, кнопка регистрации клиента, которая открывает форму регистрации, кнопка редактирования, которая открывает форму редактирования.
	Форма Редактирования и Регистрации клиента:
	1. Поле Фио.
	2. Поле Номер
	3. Поле Email

	Вкладка Номера, c созданием новера и редактирование.
	Вид вкладки: таблица со всеми номерами, кнопка создания и редактирования, которая открывает форму редактирования.
	Форма Редактирования и Создания:
	1. Поле Номер комнаты.
	2. Тип нмоера

	Вкладка Виды номеров, c созданием нвого и редактирование.
	Вид вкладки: таблица со всеми , кнопка создания и редактирования, которая открывает форму редактирования.
	Форма Редактирования и Создания:
	1. Тип номера
	2. Поле Площадь комнаты.
	3. Поле Вмещаемость людей.
	4. Поле Цена за ночь.
	5. Описание
	6. Поле Фотографий номера,
